# 認証 開発メモ

## 実装上の判断

| 判断内容 | 理由 |
|----------|------|
| `@supabase/ssr` を使用 | App Router では `@supabase/auth-helpers-nextjs` が非推奨になったため公式推奨パッケージに切り替えた |
| middleware でセッション更新を行う | クッキーの有効期限を自動延長するために必要。公式ドキュメントの推奨パターン |
| メール確認を無効化 | 開発初期のため登録フローをシンプルにする。本番前に要再検討 |

## 発生した問題と対処

| 問題 | 対処 |
|------|------|
| Server Component でセッションが取得できない | `createServerClient` に `cookies()` を渡す実装が漏れていた。公式サンプルを参照して修正 |
| middleware 適用後にログイン画面が無限リダイレクトになる | `/login` と `/register` を matcher の除外パターンに追加して解決 |
