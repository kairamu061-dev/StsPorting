# assets/

Runtime assets (textures, fonts, audio) live here and are bundled as resources
by Gradle (`sourceSets.main.resources.srcDirs = ['assets']`).

Original Slay the Spire assets are **not** included (copyrighted). The game runs
with placeholder graphics (simple shapes and the built-in libGDX font) until
free/self-made replacements are added. A Japanese-capable font is pending; UI
labels render in ASCII for now.
