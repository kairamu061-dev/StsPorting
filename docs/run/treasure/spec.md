# treasure（宝箱）仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | 中身生成 | レリック抽選＋確率で金/ポーション |
| 2 | 開封 | 中身を RunState に追加 |
| 3 | 立ち去る | 開けずに完了（分岐口） |
| 4 | 完了 | NodeResult(COMPLETED) でマップへ |

## 中身生成（方針）

- レリック：treasure 系統 RNG でプール（content/relics）から1個（既所持は除外）
- 付随報酬：一定確率で所持金（少額）またはポーション（スロット空き時）
- ポーションスロット満杯時：ポーションは付与せずスキップ（run のポーション管理に従う）

## 操作フロー

1. 宝箱ノード進入で中身生成・表示
2. 「開ける」→ `addRelic`（＋金/ポーション）→ 完了
3. 「立ち去る」→ そのまま完了
4. マップへ戻る

## エラーケース

| 条件 | 挙動 |
|------|------|
| レリックプール枯渇（全所持） | レリックなし、付随報酬のみ or 空 |
| ポーションスロット満杯 | ポーション付与スキップ |
| 既所持レリックの抽選 | 除外して再抽選 |
