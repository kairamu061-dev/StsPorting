# treasure（宝箱）概要

## 目的・背景

宝箱ノードの画面を担う。原作の「宝箱を開けてレリック（時に所持金/ポーション）を入手」を再現する。小さな画面だが独立した UI を持つためサブ項目とする。

## スコープ

### 作るもの

- **宝箱の中身生成**：RunRng（treasure 系統）でレリックを抽選。確率で所持金/ポーションが付随
- **開封演出と取得**：開ける→中身を RunState に追加
- **スキップ可**：開けずに立ち去る選択（原作の宝箱は基本開けるが、罠/呪い系宝箱の段階対応用に分岐口を残す）

### 作らないもの

- 戦闘・他ノード
- 特殊宝箱（呪い付き等）の全種（段階対応）

## 制約

- run の RunState（relics/gold/potions）を操作。
- 中身は決定的生成。

## 完了条件

- 宝箱を開けるとレリック（＋確率で金/ポーション）が RunState に入る
- 取得後マップへ戻る

## 画面イメージ

```
+-------------------------------------------+
|              📦 宝箱                       |
|   中身: [レリック] (+$30) (+ポーション?)    |
|        [ 開ける ]   [ 立ち去る ]          |
+-------------------------------------------+
```
