# shop（ショップ）仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | 在庫生成 | カード/レリック/ポーションと価格を決定的に生成 |
| 2 | 購入 | 所持金支払い→所持物追加→在庫から除去 |
| 3 | カード削除 | 価格を払いデッキ1枚削除（来店1回・価格逓増） |
| 4 | 退出 | NodeResult(COMPLETED) でマップへ |

## 在庫の構成（初期方針・原作に寄せる）

| 区分 | 点数 | 価格目安 |
|------|------|----------|
| カード | 5（コモン/アンコモン/レア混在＋アイアンクラッド/共通） | レアリティ依存（コモン安〜レア高） |
| レリック | 2–3 | 種別依存 |
| ポーション | 2–3 | 50–70 程度 |
| カード削除 | 1 サービス | 初期 75、利用ごとに +25（ラン内逓増） |

## 操作フロー

1. 商人ノード進入で在庫生成・表示
2. 商品クリック→所持金充足なら購入（`spendGold`→add）→在庫から消す
3. カード削除→デッキ一覧→1枚選択→`spendGold(removeCost)`→`removeCard`→removeCost+=25
4. 退出でマップへ

## エラーケース

| 条件 | 挙動 |
|------|------|
| 所持金不足 | 当該商品を購入不可（グレー表示） |
| カード削除を上限超過 | サービス無効化（来店1回など原作準拠） |
| デッキが削除不可（最小枚数等） | 削除サービス非活性 |
| 同一商品の二重購入 | 在庫から除去済みのため不可 |

## 未対応ケース

- メンバーシップカード/割引等の特殊効果（段階対応）
- 価格のレリック影響（段階）
