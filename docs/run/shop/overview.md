# shop（ショップ）概要

## 目的・背景

商人ノードの画面を担う。原作の「カード/レリック/ポーションの購入」「所持金を払ってのカード削除」を再現する。所持金（gold）と在庫を扱い、購入で RunState を更新する。

## スコープ

### 作るもの

- **商品陳列**：カード（数枚）・レリック（数個）・ポーション（数個）と価格
- **購入**：所持金を支払い、対応物を RunState に追加（カードはデッキ、レリック/ポーションはそれぞれ）
- **カード削除サービス**：固定/逓増価格で masterDeck から 1 枚削除
- **在庫生成**：RunRng とプール（content）から商品と価格を決定的に生成
- **退出**：いつでもマップへ戻れる

### 作らないもの

- 価格交渉/特殊割引の全種（メンバーシップ等は段階）
- 戦闘・他ノード

## 制約

- run の RunState（gold/masterDeck/relics/potions）を操作。
- 在庫は決定的生成（同一シードで同一品揃え）。
- カード削除は 1 回の来店で原作準拠の制限（通常1回、価格逓増）。

## 完了条件

- カード/レリック/ポーションを購入でき、所持金と所持物が正しく更新される
- カード削除でデッキから1枚消え、所持金が減る
- 所持金不足の商品は購入不可
- 退出でマップへ戻る

## 画面イメージ

```
+-----------------------------------------------+
|  商人        所持金: $135                      |
|  カード: [A $50][B $62][C $75]                 |
|  レリック: [R1 $150][R2 $180]                  |
|  ポーション: [P1 $50][P2 $70]                  |
|  カード削除: [ $75 ]                           |
|                                   [ 退出 ]     |
+-----------------------------------------------+
```
