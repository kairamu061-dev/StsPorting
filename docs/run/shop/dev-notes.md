# shop（ショップ）開発メモ

## 実装上の判断

| 判断内容 | 理由 |
|----------|------|
| 在庫は RunRng の shop 系統で決定的生成 | 同一シードで同一品揃え。テスト可能 |
| カード削除は価格逓増（+25）・来店制限 | 原作準拠。ラン内で removeCost を持ち越す |
| CardGridView を rest-site と共通化 | カード一覧/選択 UI の再利用で重複を避ける |
| 購入は spendGold の成否で分岐 | 所持金不足を安全に弾く |
