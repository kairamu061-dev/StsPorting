# rest-site（休憩所）概要

## 目的・背景

休憩ノード（焚き火）の画面を担う。原作の「休息（HP 回復）」「鍛冶（カードのアップグレード）」の二択を再現する。run の RunState を直接操作し、選択後にマップへ戻る。

## スコープ

### 作るもの

- **休息（Rest）**：最大 HP の一定割合（原作 30%）を回復
- **鍛冶（Smith）**：デッキからカード 1 枚を選んでアップグレード（戦闘外アップグレードの起点）
- **画面 UI**：二択ボタン、鍛冶時のデッキ表示とカード選択
- **レリックによる選択肢変化**：（段階）休憩所追加効果のフック点

### 作らないもの

- 戦闘・他ノード
- 鍛冶以外のデッキ操作（削除等はショップ/イベント）
- 焚き火の装飾演出（最小再現）

## 制約

- run の RunState（currentHp/maxHp/masterDeck）を操作。
- カードアップグレードは combat/cards の upgrade() を masterDeck 上のインスタンスに適用。
- 1 回の休憩で休息か鍛冶のどちらか 1 つ（原作準拠）。

## 完了条件

- 休息で HP が最大の 30% 回復（最大超過はクランプ）
- 鍛冶でデッキから 1 枚をアップグレードできる（既に最大は対象外）
- 選択後マップへ戻る

## 画面イメージ

```
+-------------------------------------------+
|            🔥 休憩所                       |
|                                           |
|     [ 休息：HP +30% ]                      |
|     [ 鍛冶：カード強化 ]                   |
+-------------------------------------------+
鍛冶選択時 → デッキ一覧からアップグレード対象を選択
```
