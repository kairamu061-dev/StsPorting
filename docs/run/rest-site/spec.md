# rest-site（休憩所）仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | 休息 | 最大HP の 30%（切り上げ/原作準拠）を回復 |
| 2 | 鍛冶 | デッキ1枚をアップグレード |
| 3 | カード選択 | 鍛冶時にデッキ一覧から対象選択（最大強化済みは除外） |
| 4 | 完了 | 選択後 NodeResult(COMPLETED) を返しマップへ |

## 操作フロー

1. 休憩ノード進入で焚き火画面表示
2. 「休息」or「鍛冶」を選択
3. 休息：`RunState.heal(ceil(maxHp*0.3))` → 完了
4. 鍛冶：デッキ一覧表示 → カード選択 → `card.upgrade()` → 完了
5. マップへ戻る

## エラーケース

| 条件 | 挙動 |
|------|------|
| 鍛冶でアップグレード可能カードが0 | 鍛冶を選べない/休息のみ |
| 既に最大HP | 休息しても回復0（選択は可、原作準拠） |
| 鍛冶選択後にキャンセル | 二択画面へ戻る（未確定なら消費しない） |

## 未対応ケース

- レリック/特殊効果による休憩追加選択肢（段階対応）
- 焚き火イベント（一部レリックの特殊休憩）
