# monsters（content）開発メモ

## 実装上の判断

| 判断内容 | 理由 |
|----------|------|
| 第1幕の「AI/インテントの代表」を選定 | 単純固定（Cultist）・混合抽選（JawWorm）・行動反応（GremlinNob/Enrage）・状態遷移（Lagavulin）・フェーズ（Guardian）を最小数で網羅 |
| Enrage はプレイヤーの onUseCard を購読する形 | 「スキル使用で敵が強化」を powers のフックで表現。敵→プレイヤーイベント接続を combat/powers と擦り合わせ |
| ボスはまずモードシフトの基本挙動を再現 | 第1幕完走目標に対し、フェーズ切替の代表を1体で成立させる |
| HP・行動は RunRng で決定的 | 同一シードで同一展開、テスト可能 |
