# monsters（content）仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | 通常敵定義 | HP・moves（候補/重み/連続上限）・takeTurn |
| 2 | エリート定義 | 行動反応 AI / 状態遷移を含む強敵 |
| 3 | ボス定義 | フェーズ/特殊行動を持つ第1幕ボス |
| 4 | エンカウンター | ノード種別ごとの編成テーブルと抽選 |
| 5 | MonsterLibrary | MonsterId → 生成・登録 |

## 各敵の行動仕様（代表）

### カルティスト Cultist（通常）
- 初手：儀式（Ritual）= 自身に「毎ターン筋力+n」のバフ付与（DEBUFF ではなく BUFF）
- 以降：攻撃（中ダメージ）を継続
- AI：1ターン目固定で儀式 → 2ターン目以降は攻撃固定（連続制限不要の単純パターン代表）

### ジョー・ラガヴリン Jaw Worm（通常）
- 候補：強攻撃 / 防御＋筋力 / 防御。重み付き抽選＋連続制限
- AI：攻撃・防御・バフが混ざる代表

### グレムリンノブ Gremlin Nob（エリート）
- 「怒り（Enrage）」：プレイヤーがスキルをプレイするたび筋力増（onUseCard 反応の代表）
- 強攻撃を周期的に。スキル多用を咎める AI
- 実装：Enrage はプレイヤーの onUseCard を購読するパワーとして敵に付与

### ラガヴリン Lagavulin（エリート）
- 開幕数ターン睡眠（防御態勢・行動せず or 高ブロック）→ 一定後に覚醒し攻撃＋デバフ（筋力/敏捷低下系）
- 状態遷移（眠り→覚醒）の代表

### ザ・ガーディアン The Guardian（ボス・初期実装候補）
- モードシフト：一定ダメージを受けると防御モード（高ブロック・とげ的反撃）へ移行、解除でオフェンスモードへ
- フェーズ切替の代表。まずはモードシフトの基本挙動を再現

## エンカウンター定義

```
EncounterTable {
  act1_weak:   [ {Cultist}, {JawWorm}, {2x small} ... ]   // 序盤の弱編成
  act1_strong: [ {3x small}, {mixed} ... ]                // 中盤の通常編成
  act1_elite:  [ {GremlinNob}, {Lagavulin} ]
  act1_boss:   [ {TheGuardian} ]
}
rollEncounter(NodeType, act, rng): EncounterDef
```

- マップ生成時に各モンスター/エリート/ボスノードへ、序盤/中盤の別と RNG で編成を割り当てる（重複しすぎない原作の配慮は段階的に）。

## エラーケース

| 条件 | 挙動 |
|------|------|
| 候補が全て連続上限で除外 | フォールバック行動（各敵に既定 move を定義） |
| ボスのモードシフト境界 | 受けたダメージ累計でしきい値判定。複数回切替に対応 |
| 複数体の同時行動 | 配置インデックス順に takeTurn |
| HP ロールが範囲外 | 範囲内にクランプ（決定的） |

## 未対応ケース

- 全敵・他幕・分裂/召喚の高度ギミック
- 原作の厳密な確率テーブルの完全一致（段階的に近づける）
