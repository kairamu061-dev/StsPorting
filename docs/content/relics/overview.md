# relics（content）概要

## 目的・背景

レリック（永続アイテム）を定義する。レリックは常時/取得時/戦闘中の各タイミングでフックを発火し、プレイに恒常的な影響を与える。combat のフック点（ターン頭/被弾時/カードプレイ時等）と run の取得時フックに接続する。

全レリック網羅はせず、「初期レリック＋フック種類の代表」を再現対象とする（量より仕組みの網羅）。

## スコープ

### 作るもの

- **初期レリック**：バーニングブラッド（戦闘終了時 HP+6）
- **代表フックレリック**：
  - 取得時即時系（例：最大 HP 増、初期ポーション）
  - ターン頭系（例：毎ターン最初にブロック/エネルギー）
  - 戦闘開始系（例：開幕にバフ/ブロック）
  - カードプレイ反応系（例：ターン中 N 枚目で効果）
  - 被弾/攻撃反応系
- **RelicLibrary**：RelicId → AbstractRelic 生成・登録
- **レリック取得処理連携**：run の addRelic からフック発火

### 作らないもの

- 全レリック網羅
- 他キャラ専用レリック
- レリックアイコン画像（プレースホルダ）

## スコープ補足（再現対象の初期候補）

| レリック | 種別/入手 | 効果（フック） |
|----------|-----------|----------------|
| バーニングブラッド Burning Blood | 初期 | 戦闘終了時 HP+6（onVictory） |
| アンセインドハート | （例）ボス初期相当 | （段階） |
| ブラッドバイアル Blood Vial | スターター級 | 取得時：最大HP は変えず、戦闘開始時 HP+2（atBattleStart） |
| アンカー Anchor | コモン | 戦闘開始時 ブロック+10（atBattleStart、初回ターンのみ） |
| ブロンズの鱗 Bronze Scales | コモン | とげ3 を常時（戦闘開始時に Thorns 付与） |
| ヴァジュラ Vajra | コモン | 戦闘開始時 筋力+1 |
| オッドリースムーズストーン | コモン | 戦闘開始時 敏捷+1（または相当） |
| ペンナイブ/Pen Nib 系 | アンコモン | カードを N 回プレイで次の攻撃 2 倍（onUseCard カウント） |
| 取得時HP系（例：ストロベリー/ペアー） | フード | 取得時 最大HP+ |

> 「初期＋戦闘開始バフ＋常時とげ＋カードプレイカウント＋取得時HP」でフック種類を代表。原作効果に寄せて実装。

## 制約

- combat の AbstractRelic フックモデルに接続（atBattleStart/atTurnStart/onUseCard/onAttacked/onVictory 等）。
- 取得時フックは run の addRelic から発火。
- 量より「フック種類の網羅」を優先し、段階的に追加。

## 完了条件

- バーニングブラッドが戦闘終了時に HP+6 する
- 戦闘開始系/常時系/カードプレイ反応系の代表が各1つ以上機能する
- 取得時系レリックが run で取得時に発火する
- レリック効果のユニットテストが通る
