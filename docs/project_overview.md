# プロジェクト概要

## 目的・背景

『Slay the Spire』（以下 STS）のデスクトップ版の挙動を、Windows デスクトップ上で再現するポーティングプロジェクト。
原作は Java + libGDX で実装されており、本プロジェクトも同一エンジンを採用することで、フレーム単位の挙動・エフェクト・入力タイミング・操作感を忠実に再現することを狙う。

学習・検証を目的とした再現実装であり、商用配布や原作アセットの二次配布は行わない。

## スコープ

### 作るもの

- アイアンクラッド（Ironclad）1 キャラクターでの通常プレイ（ニュートラルラン）
- 1 幕〜（再現可能な範囲で）進行できるシングルプレイのローグライク戦闘・マップ進行
- カードプレイ、戦闘システム、状態異常（バフ／デバフ）、エネルギー、ドロー／捨て札／消滅などのコアシステム
- マップ選択、ショップ、休憩、宝箱、イベント、ボス戦などのノード進行
- **操作性・システム・エフェクト・演出の完全再現**（これが本プロジェクトの最優先要件）

### 作らないもの

- アイアンクラッド以外のキャラクター（サイレント／ディフェクト／ウォッチャー）
- デイリークライム、カスタムモード、シード共有などの特殊モード
- アセンション（昇格）難易度
- 全カード／全敵／全レリック／全ポーション／全イベントの網羅的再現（コア体験に必要な範囲に絞る）
- マルチプレイ、実績、ランキング等のオンライン機能

### あれば良い（任意）

- オプション画面（設定）
- プレイ履歴

## 技術スタック概要

| 技術 | 用途 | 選定理由 |
|------|------|----------|
| Java | 実装言語 | 原作と同一。デコンパイル資料が全て Java で参照しやすい |
| libGDX | ゲームエンジン | 原作と同一エンジン。描画・入力・タイミング・エフェクトを忠実に再現可能 |
| Gradle | ビルド／依存管理 | libGDX 標準のビルドツール |
| jpackage / jlink | Windows 配布 | JRE 同梱の Windows 実行ファイル（.exe）を生成 |

## 全体アーキテクチャ

- **エンジン層**: libGDX（描画・入力・オーディオ・ファイル I/O）
- **ゲームループ層**: `Game` / `Screen` によるシーン管理（メインメニュー → ゲーム → 戦闘 → マップ ...）
- **ドメイン層**: カード・クリーチャー（プレイヤー／敵）・状態異常（パワー）・レリック・ポーション・アクション（行動キュー）
- **アクションキュー**: 原作同様、戦闘中の処理を逐次実行する `ActionManager` 相当のキューで、演出と状態変更の順序・タイミングを再現
- **UI / エフェクト層**: カード描画、ダメージ数値、パーティクル、画面エフェクトなど

機能エリア単位の詳細は各 `overview.md` / `design.md` を参照（[index](./index.md)）。

## 制約

- 開発環境は Linux（WSL2 / devcontainer）。GUI を伴う実機確認は Windows 側で行う必要がある。
- 現状この環境には JDK / Gradle が未導入（セットアップが必要）。
- 原作のアセット（画像・音声・カードデータ）は著作物のため同梱しない。プレースホルダまたは自作アセットで代替し、形状・タイミングの再現に注力する。
- 「完全再現」の対象は操作性・システム・エフェクト・演出であり、コンテンツ量（カード／敵の数）は対象外。

## UI デザイン方針

### コンセプト

原作 STS のダークファンタジー基調の UI を踏襲する。盤面（戦闘画面）・手札の弧状レイアウト・カード操作のドラッグ感を忠実に再現する。

### カラーパレット

| 用途 | カラーコード | 説明 |
|------|-------------|------|
| プライマリ | `#C8AA6E` | STS 風の金／琥珀色アクセント |
| セカンダリ | `#8B0000` | アイアンクラッド基調の暗赤 |
| 背景 | `#1A1410` | 暗い洞窟調の背景 |
| テキスト | `#F0E6D2` | 羊皮紙色のテキスト |
| エラー | `#E03A3A` | 警告・エラー表示 |

### タイポグラフィ

原作に近いセリフ系フォントを基本とする（同梱可能なオープンフォントで代替）。数値表示は視認性の高い等幅／太字系を使用。

### コンポーネント方針

- カード: 弧状に並ぶ手札、ホバーで拡大、ドラッグでプレイ。原作の挙動・アニメーションを再現
- ボタン: 羊皮紙／石板調。ホバー・押下のフィードバックを付与
- 戦闘 HUD: エネルギー球、終了ターンボタン、山札／捨て札カウンタを原作配置で再現
